# Vagrant Hands On

## Initialize `Vagrantfile`

```bash
vagrant init BOX_NAME
```

## Create/start VM

```bash
vagrant up
```

## Get VMs status

```bash
vagrant status
```

## Get global VMs status

```bash
vagrant global-status --prune
```

## Reload VM

```bash
vagrant reload
```

## Shutdown VM

```bash
vagrant halt
```

## Destroy VM

```bash
vagrant destroy
```

## Show SSH config

```bash
vagrant ssh-config
```

## Access using SSH

```bash
vagrant ssh
```
